How RBAC Works

The system checks permissions in this order:

  1. First, it checks if you have a specific user permission assigned
  2. If not, it checks if your role has the permission
  3. If neither exists, access is denied
Role Permissions

Each role has predefined permissions:

  • Admin: All permissions
  • Manager: Most permissions except user management
  • Cashier: Limited permissions focused on sales
  • Stock Manager: Inventory-related permissions
